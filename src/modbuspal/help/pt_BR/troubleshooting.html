Troubleshooting (pt-BR)

Voltar ao índice

Cannot bind port 502

  • Outra aplicação está usando a porta 502.
  • Troque a porta TCP ou finalize o processo concorrente.

Erro RXTX / serial indisponível

  • Ambientes Java modernos podem não expor java.ext.dirs.
  • Nesse caso, utilize TCP/IP ou Replay.
  • Use Tools > Check dependencies para validar rapidamente o estado.

Projeto não carregou corretamente

  • Valide se o XML .xmpp está íntegro e compatível.
  • Confira se scripts referenciados existem no caminho relativo esperado.
