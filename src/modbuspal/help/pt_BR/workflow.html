Voltar ao índice

Passo 1 - Carregar projeto

Menu Project > Carregar projeto e escolha um arquivo .xmpp.

Passo 2 - Validar link

Em Link settings, selecione TCP/IP, Serial ou Replay conforme cenário didático.

Passo 3 - Habilitar planta

Use os controles de slaves e automações para colocar os modelos em execução.

Passo 4 - Iniciar simulação

Clique em Run, depois use Master para leitura/escrita de setpoints.

Passo 5 - Salvar e encerrar

Menu Project > Salvar projeto, depois Sair para preservar estado.
