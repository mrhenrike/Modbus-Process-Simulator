Operational Workflow (en-US)

Back to index

Step 1 - Load project

Open Project > Load project and select a .xmpp file.

Step 2 - Validate link mode

Choose TCP/IP, Serial, or Replay in Link settings.

Step 3 - Enable plant model

Enable slaves and start automations for your scenario.

Step 4 - Start simulation

Click Run, then use Master to read/write setpoints.

Step 5 - Save and exit

Save with Project > Save project, then close via Exit.
