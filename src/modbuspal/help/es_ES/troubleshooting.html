Troubleshooting (es-ES)

Volver al índice

Cannot bind port 502

  • Otro proceso ya está usando el puerto 502.
  • Cambie el puerto TCP o cierre el proceso en conflicto.

RXTX / serial no disponible

  • En Java moderno puede no existir java.ext.dirs.
  • Use TCP/IP o Replay cuando serial no esté disponible.
  • Abra Tools > Verificar dependencias para diagnóstico rápido.

Problemas al cargar proyecto

  • Valide integridad y compatibilidad del archivo .xmpp.
  • Confirme que los scripts referenciados existen en rutas relativas válidas.
